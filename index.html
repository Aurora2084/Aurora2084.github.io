线性代数第一讲，线性空间
数学基础 2025/7/9

线性空间

线性空间与子空间

$ \mathbb{R}^3 $上的直线可以用两个常数向量的线性组合表示，类似的，如果一个集合中可以定义“加法”和“数乘”两个运算，就称其为线性空间。

定义，线性空间

一个数域 k($k=\mathbb{R}$ 或者$k=\mathbb{C}$ )上的线性空间(或称向量空间)指的是一个集合 V 以及二元运算
$$\text{加法: } V \times V \xrightarrow{\oplus} V \quad \text{数乘: } k \times V \rightarrow V$$
使得对任何 $u,v,w\in V,a,b\in k$,下列性质成立:

  1. 加法交换律: $u+v=v+u$
  2. 加法结合律: $(u+v)+w=u+(v+w)$
  3. 零元素: 存在唯一元素 $0\in V$ 使得 $0+u=u$
  4. 负元素: 存在唯一的 $−u\in V$ 使得 $u+(−u)=0$
  5. 单位元: 对 $k$ 中的单位元 1, 满足 $1u=u$
  6. 数乘结合律: $a(bu)=(ab)u$
  7. 分配律(向量加法): $a(u+v)=au+av$
  8. 分配律(数量加法): $(a+b)u=au+bu$
    线性空间 V 中的元素称为一个向量。
    如不特殊说明, 我们考虑实线性空间(即 $ k=\mathbb{R} $ )。

易知， $ \mathbb{R}^3 $ 是一个线性空间

例子，所有一元多项式组成的集合是线性空间 $ \mathbb{R}[x] $，所有一元光滑函数构成的空间 $ C^{ \infty}( \mathbb{R}) $

直和

定义，线性空间

对于给定的线性空间 $ V,W $
$$ V \oplus W= { (v,w)|v \in V, w \in W} $$
这个运算称为直和，得到一个新的线性空间。

加法 $ (v_1,w_1)+ (v_2,w_2)= (v_1+v_2,w_1+w_2) $

数乘 $ \lambda (v,w)=( \lambda v, \lambda w) $

类似的可定义n项直和， $ V_1 \oplus V_2 \oplus \cdots \oplus V_m $,
直和中的元素可以表示为一向量$ (u_1, \cdots, u_i),u_i \in V_i $

易知, $ \mathbb{R}^n $是n个 $ \mathbb{R} $的直和
,$ \mathbb{R}^n= \mathbb{R}^m + \mathbb{R}^k,n=m+k $

线性子空间

定义，线性子空间

$ V $是数域k(实数，复数等)上的线性空间， $ V $ 的一个子集 $ W $是 $ V $ 的一个线性子空间。

$ W $需要包括 $ V $ 的零元素，且 $ W $ 的元素在加法和数乘下封闭（即运算后的元素仍在 $ W $内 ）

$ V $的线性子空间是线性空间 　

例子，过原点的直线是 $ \mathbb{R}^2 $的一个线性子空间，不过原点的直线不包含零元素（0,0）

线性组合与线性相关

线性组合

线性空间中的元素可以由更简单的元素通过线性组合得到，如三维矢量由坐标点给出

定义 ,线性组合

设 $\mathbf{v}_1, \cdots, \mathbf{v}_m \in V$, $\lambda_1, \cdots, \lambda_m \in k$. 则向量
$$\mathbf{u} = \lambda_1 \mathbf{v}_1 + \cdots + \lambda_m \mathbf{v}_m$$
称为向量 $\mathbf{v}_1, \cdots, \mathbf{v}_m$ 的线性组合, 或者称 $\mathbf{u}$ 可以由 $\mathbf{v}_1, \cdots, \mathbf{v}_m$ 线性表达. 所有可以由 $\mathbf{v}_1, \cdots, \mathbf{v}_m$ 线性表达的向量构成的集合记为
$$
\operatorname{Span}{\mathbf{v}_1, \cdots, \mathbf{v}_m}$$

命题 $\operatorname{Span}{\mathbf{v}_1, \cdots, \mathbf{v}_m}$ 是 $V$ 的线性子空间.

例子1， $ \mathbb{R}^3 $中两个常矢量线性组合张成的平面记作 $ \operatorname{Span} { \vec{u}, \vec{v} } $
例子2，微分方程 $ f’’(x)+f(x)=0 $的解系为 $ \operatorname{Span}{ \cos x,sin x }$

为了分析线性表示的唯一性，需要引入线性相关的概念

线性相关性

定义

$ V $中的若干元素是线性相关的，则有不全为零的系数使得
$$ \lambda _{1} v_1+ \cdots + \lambda _{m}v_m=0 $$
否则这些元素线性无关

命题，线性相关的一组向量中其中一个向量可以被其他向量线性表示

可知，从线性相关向量组去除一个向量后，其余的向量不再线性相关，我们可以理解为线性相关向量组包括了冗余的线性信息，可以描述其中的元素。

线性无关性

一组向量线性无关的等价表示为，这组向量只有在系数全为0时才能相加为0。

例子 $ \mathbb{R}^3 $上的一组正交基是线性无关的

迁移与扩散引起的物质传递

一般物质传递公式

流量与电化学梯度有关

$$ \vec{J_j}=-( \frac{ C_j D_j }{ RT } )\nabla \overline{ \mu_j} $$

在一维上， $ j_j(x)=-( \frac{ C_j D_j }{ RT }) \frac{ \partial { \overline{ \mu_j}} }{ \partial {x} } $ ,如果溶液在流动，则增加 $ C_jv(x) $项

当 $ a_j \approx C_j $ 时，得到能斯特-普朗克方程

$$ J_j(x)=-( \frac{ C_jD_j }{ RT } )[ \frac{ \partial }{ \partial {x} } (RT \ln{C_j })+ \frac{ \partial }{ \partial {x} } (z_j F \phi) ]+C_jv(x) $$

$$ j_j(x)= -D_j \frac{ \partial { C_j(x) }}{ \partial {x} } - \frac{ z_jF }{ RT } D_jC_j \frac{ \partial { \phi(x)} }{ \partial {x} } + C_jv(x) $$

不考虑对流，物质j带电荷，流量 $ J_j $ 等价于电流密度，有

$$ -J_j= \frac{ i_j }{ z_jFA } = \frac{ i_{d,j} }{ z_j FA} + \frac{ i_{m,j} }{ z_jFA } $$

$ i_{d,j} ， i_{m,j} $ 分别为扩散和迁移电流
$$ \frac{ i_{d,j} }{ z_jFA } =D_j \frac{ \partial {C_j} }{ \partial {x} } $$

$$ \frac{ i_{m,j} }{ z_jFA } = \frac{ z_j FD_j }{ RT } C_j \frac{ \partial { \phi} }{ \partial {x} } $$

总电流i为所有物质贡献组成

$$ i= \sum _{j }i_j $$

$$ i= \frac{ F^2A }{RT } \times \frac{ \partial { \phi} }{ \partial {x} }\sum_j z_j^2D_jC_j+FA\sum z_jD_j \frac{ \partial {C_j} }{ \partial {x} } $$

迁移

在本体溶液中，浓度梯度较小，只考虑迁移项

$$ i= \frac{ F^2A }{RT } \times \frac{ \partial { \phi} }{ \partial {x} }\sum_j z_j^2D_jC_j$$

定义淌度 $ \mu $

$$ \mu_j= \frac{ z_jFD_j }{ RT } = \frac{ |z_j|e }{ 6 \pi \eta r } $$

对于线性电场 $ \frac{ \partial { \phi} }{ \partial {x} } = \frac{ \Delta E }{ l } $ ,

$$ i= \frac{ FA \Delta E }{l } \sum _{j} |z_j| \mu_j C_j $$

$$ L= \frac{ 1 }{ R } = \frac{ A }{ l } \kappa $$

$$ \kappa =F \sum _{j} |z_j| \mu_j C_j $$

指定离子的电流在总电流中的分数为迁移数$t_j$

$$ t_j= \frac{ i_j }{ i } = \frac{ |z_j| \mu_j C_j }{ \sum _{k}|z_k| \mu_k C_k }= \frac{ |z_j| \lambda_j C_j }{ \sum _{k}|z_k| \lambda_k C_k } $$
