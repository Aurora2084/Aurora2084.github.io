材料的力学性能

仅供学习交流使用，内容来自2025年秋季学期。本人不对本文中的图片保有版权。

公式没渲染出来的话刷新一下网页就好了。

受力形变

形变，就是材料的形状和体积对于外力的响应。

  • 脆性材料：只有弹性变形阶段，几乎不发生塑性变形，超过弹性极限后断裂。绝大多数无机材料是这种变形行为，总弹性形变能很小
  • 延性材料：开始为弹性变形，接着有一段塑性变形，总变形能很大。如低碳钢。
  • 弹性材料：具有极大的弹性形变，没有残余形变。如橡胶

当作用力较小时，$\sigma<\sigma_s$,发生可逆的弹性形变；作用力超过$\sigma_s$,则发生不可逆的塑性形变。高温恒应力下发生蠕变。

基本力学行为

受力$\to$ 弹性形变$\to$ 塑性形变$\to$ 断裂

基本力学指标包括强度，塑性，韧性。

  • 屈服强度 $\sigma_s$
  • 抗拉强度 $\sigma_b$
  • 延伸率$\delta$
  • 断面收缩率$\psi$

应力

$$\sigma=\frac{F}{A}$$

名义应力：A选用受力前的初始面积。实际一般用名义应力。

真实应力：A选用受力后的真实面积。

规定：

  1. 应力分量下标第一个字母表示作用面的法向方向，第二个字母表示应力作用的方向。法向应力可以略去一个。
  2. 拉应力（产生正应变）为正，压应力为负

应变

物体内部各质点相对位移量
$$\varepsilon=\frac{\Delta L}{L}$$

法向应变

$\varepsilon$称为名义应变。真实应变为$\varepsilon_{true}=\ln\frac{L_1}{L_0}$
一般都使用名义应变。

剪应变

剪应力作用下产生的应变，是体积元上两个面夹角的变化。

压缩应变

受到流体的静压力作用而产生压缩应力，压缩应变$\Delta=\frac{\Delta V}{V_0}$

低碳钢的典型$\sigma-\varepsilon$曲线

弹性形变

外力去除后形变完全消失的称为弹性形变。

  • 可逆性
  • 线性（单值，胡克定律）
  • 变形量较小
    高分子材料的弹性形变一般都为非线性的。而常见金属的弹性形变和结晶态高分子的弹性变形量很小（1%）。

单向应力状态

各向同性的长方体沿x轴受到拉力
$$\varepsilon=\frac{\Delta L}{L}=\frac{\sigma_x}{E}$$
E为弹性模量

横向发生收缩，引入横向变形系数$\mu$,称为泊松比。
$$\mu=\vert\frac{\varepsilon_y}{\varepsilon_z}\vert=\vert\frac{\sigma_z}{\sigma_x}\vert$$
金属材料的泊松比约为0.3，陶瓷约为0.2。

三向应力状态

各方向总应力是分量的线性叠加。
$$\varepsilon_1=\frac{\sigma_1}{E}+-\mu\frac{\sigma_2}{E}+-\mu\frac{\sigma_3}{E}$$

此时胡克定律即为
$$\varepsilon_x=\frac{1}{E}[\sigma_x-\mu(\sigma_y+\sigma_z)]$$
另外两个方向同理

剪切应变

$$\gamma_{xy}=\frac{\tau_{xy}}{G}$$
G为剪切模量
$$G=\frac{E}{2(1+\mu)}$$
对于各向异性的材料，不满足胡克定律。而要用广义胡克定律描述。

弹性变形的本质

在外力作用下，构成材料的原子从平衡位置产生可逆位移

弹性模量的本质

弹性模量E越大，材料刚性越强，越不易产生变形。
从原子尺度看，弹性模量是原子键合强度的标志，与原子间结合力曲线的一阶导数有关。

弹性模量表征材料对弹性变形的抵抗力。工程上称为刚度。但是弹性模量是材料的本征参数，而实际工件的抗变形能力还与尺寸和载荷等有关，因此二者不能简单等同。

弹性模量的影响因素

  • 键合方式：键合方式越强，弹性模量越大。
  • 温度升高，原子间距增大，作用力减小，弹性模量降低
  • 应力：压应力使得原子间距减小，弹性模量增大。张应力使得原子间距增大，弹性模量减小。
  • 化学组成
    • 合金的弹性模量随组成元素的质量分数、组织结构、晶体结构的变化而变化
    • 固溶体合金的E主要取决于溶剂元素的性质和晶体结构。
  • 微观组织
  • 金属的弹性模量对组织不敏感，钢的热处理能显著提高强度，但是E变化不是很大
  • 陶瓷的弹性模量随气孔率增高下降，经验公式有
    $$E=E_0(1-1.9p+0.9p^2)$$,p 为气孔率
  • 高分子材料的弹性模量可通过添加剂提高

实际多组元体系的弹性模量介于高弹性模量和低弹性模量之间。
假定体系内应力均匀

  1. 两相系统泊松比和应变相同，得到上限弹性模量
    $$E_U=E_1V_1+E_2V_2$$
    $V_i$为对应的体积分数

2.假定泊松比和应力相同，得到下限弹性模量
$$\frac{1}{E_L}=\frac{V_1}{E_1}+\frac{V_2}{E_2}$$

  • 原子结构
    -金属元素的原子半径越大，E越小
    • 过渡元素的弹性模量较大，因为其原子半径较小，且d层电子引起较大的原子间结合力
  • 晶体结构
    • 单晶的E各向异性
    • 多晶的E伪各向同性
    • 玻璃态的E各向同性

粘弹性和滞弹性

实际材料的与时间有关的弹性成为滞弹性。应变滞后于应力。

可以用弹性元件和粘性元件的组合模拟这种力学行为。

弹性元件$\sigma=E\varepsilon$,粘性元件$\sigma=\eta \frac{\mathrm{d}\varepsilon}{\mathrm{d}t}$

麦克斯韦模型

麦克斯韦液体是液态粘弹性液体，由弹簧和粘壶串联

应用：应力松弛。$\varepsilon=\varepsilon_1+\varepsilon_2$

开尔文固体模型

描述固体粘弹性行为，并联，$\sigma=\sigma_1+\sigma_2$
应用：蠕变（应变松弛）

一些材料（如高温下的玻璃或高分子）在小应力下同时表现出粘性和弹性，称为粘弹性。可以认为是充分发展的滞弹性。

松弛

应变松弛（蠕变）

材料在恒应力下变形缓慢增加，如沥青等的膨胀。（并联模型）
$$E_c(t)=\frac{\sigma_0}{\varepsilon(t)}$$

应力松弛（弛豫）

在持续外力下发生变形，总变形量不变，由于粘性的徐变变形增加，使得弹性变形减小，应力下降。（串联模型）
$$E_{\tau}=\frac{\sigma(t)}{\varepsilon_0}$$
一般来说，弛豫的来源是玻璃相

塑性形变

材料经过塑性形变而不失效的能力称为延展性，是重要的力学性能。

指标：

  • 断裂伸长率（延伸率）：$\delta=\frac{\Delta l}{l_0}\times 100%$
  • 断面收缩率 ：$\psi=\frac{\Delta A}{A_0}$

只有极少数离子晶体在常温下有延展性，如AgCl可以冷轧，KCl晶体可以弯曲。他们都属于NaCl型结构。

四方$ZrO_2$晶体掺有少量$CeO_2$后在一定压力下发生相变（转为单斜晶系）而出现很大的塑性变形能力，称为超塑性或相变塑性。

滑移在晶体中更常见，对变形的贡献更大。
滑移只发生在特定的晶面和晶向上。

滑移面和滑移方向一般为密排面和密排方向。

滑移系数目越多，塑性越好

滑移方向的权重更高，所以fcc的塑性更好。

多晶材料不易滑移，晶界对于滑移有钉扎作用。


法线上应力$\sigma=\frac{F}{A}\cos^2\varphi$,滑移方向上剪应力为$\tau=\frac{F}{A}\cos\lambda\cos\varphi$

剪应力大于临界值发生滑移

塑性形变指数

高温蠕变

材料在高温和低于屈服强度的应力下随时间产生塑性变形。

高温

  • 金属 $T>0.3\sim 0.4 T_m$,$T_m$为熔点
  • 陶瓷 $T>0.4\sim 0.5 T_m$
  • 高分子 $T>T_g$,大于玻璃化转变温度

  1. 起始阶段：普通线弹性
  2. 减速阶段，应变速率减小$\frac{\mathrm{d}\varepsilon}{\mathrm{d}t}=At^{-n}$
  3. 稳态阶段，蠕变速率几乎不变$\varepsilon=Kt$
  4. 加速阶段，蠕变曲线变陡，最后失效

外场增加曲线变陡。

高温蠕变理论

蠕变机理包括晶格蠕变和晶界蠕变。

晶格蠕变

1.位错主导 长程应力下的位错滑移导致的蠕变

易运动的位错解放出来后被钉扎，即为减速阶段；应力大到受到较大阻碍的位错也开始释放，即为加速阶段。

2.扩散主导

点缺陷（空穴、间隙原子）扩散，属于短程应力场的扩散蠕变

  • 体扩散，在晶粒内部
  • 晶界扩散，在晶界进行

晶界蠕变

非晶态晶界在高温下发生流动
1.晶界占比高，高温下近似牛顿流体
2.晶界占比低，蠕变主要变形为晶界滑移和晶粒本身的变形

  • 扩散发生在内部称为N-H蠕变
  • 扩散发生在晶界称为Coble蠕变
    3.晶界占比低，晶粒几乎不形变的情形，不能视为牛顿流体，其描述较为复杂
蠕变断裂
  • 晶格蠕变 $\to$ 穿晶断裂
  • 晶界蠕变 $\to$ 沿晶断裂

影响蠕变的因素

1.温度：温度升高蠕变速率增大
2.应力，剪应力增大蠕变速率、压应力减小蠕变速率。
3.显微结构，气孔、晶粒尺寸、玻璃相等


这是由于玻璃相是可流动的部分，润滑了晶粒。

4.组成 不同的化学计量数比影响了蠕变性能。
5.晶体结构：共价键比例高，蠕变能力减弱

超塑性

1.细晶超塑性：晶界滑动是主要的变形方式
2.相变超塑性
3.其他

细晶结构是超塑性的先决条件

变形机制

1.准均匀流变，各晶粒均匀形变

2.非均匀流变

  • 晶内位错滑移
  • 晶界位错滑移

材料的断裂与强度

韧性断裂

结构发生大量变形、原始尺寸明显变化。

塑性形变和粘性形变导致的永久变形，是断口处应力超过了屈服强度导致的。断口处呈灰黑色纤维状。

脆性断裂

不发生明显变形，突然发生。断口光滑呈结晶状。

脆性断裂是由于应力集中处的缺陷扩展引起的。

理论断裂强度

Orowan近似

基于正弦波拟合原子间作用力。
理论结合（断裂）强度
$$\sigma_{th}=\sqrt{\frac{E\gamma}{a}}$$

E为弹性模量，$\gamma$为断裂表面能，a为晶格常数。
对于一般材料，近似有$\sigma_{th}=\frac{E}{10}$

实际材料的强度比理论值低得多，这是因为断裂在缺陷处扩展更快。

微裂纹理论

Inglis理论

顶端应力
$$\sigma_A=\sigma[1+2\sqrt{\frac{c}{\rho}}]$$

扁孔顶部发生应力集中，近似有

$$\sigma_A=\sigma_{th}$$
$$2\sigma \sqrt{\frac{c}{\rho}}=\sqrt{\frac{E\gamma}{a}}$$
临界应力
$$\sigma_c=\sqrt{\frac{E\gamma}{4c}}$$

格里菲斯断裂理论

从能量的角度考虑裂纹扩展。

临界应力
$$\sigma_c=\sqrt{\frac{2E\gamma}{\pi c }}$$
与$\sigma_{th}$比较，如果裂纹长度在a尺度，则可接近理论强度

对于塑性材料，塑性形变将消耗微裂纹扩展的能量，格里菲斯理论不再适用

裂纹的扩展

$$\sigma_c=Kc^{-\frac{1}{2}}$$

裂纹尖端的应力场

研究I型裂纹的应力分布

定义应力场强度因子$K_1$,单位$Pa\cdotm^{\frac{1}{2}}$,与外加应力与受力状态，裂纹长度和种类有关。

定义几何形状因子$Y$,与试件和裂纹的几何形状有关,$K_1=Y\sigma \sqrt{c}$

临界应力场强度因子

根据第一强度理论$\sigma \leq [\sigma]$，这种方法不能防止低应力下的脆性断裂。

根据断裂力学观点，$K_1=Y\sigma \sqrt{c}>K_{Ic}$时裂纹扩展，极限值称为断裂韧性。

裂纹扩展的动力和阻力

动力

单位面积裂纹扩展释放的弹性应变能
$$G=-\frac{\mathrm{d}W_E}{\mathrm{d}A}$$
称为裂纹扩展力。

阻力

$$K_{Ic}=\sqrt{2E\gamma}$$

裂纹的起源

缺陷处的应力集中

化学或机械损伤

热应力导致开裂

裂纹可以分为本征裂纹（气孔、夹杂、挥发物的失去、异常长大晶粒、各向异性的热膨胀或弹性形变）和非本征裂纹（外力引入的）

减小工作应力，加入塑性相。引入小于临界尺寸的裂纹能够有效增韧。

裂纹亚临界生长

在低于临界应力下工作较长时间仍会导致材料破坏。

应力腐蚀理论

陶瓷的应力空腔

亚临界裂纹生长速率与应力场强度因子

裂纹生长速率$v$随$K_1$的增大而增大

高温延迟断裂

高温下材料在低于临界应力的外力作用后，在外力持续一段时间后断裂，一般是固有裂纹在外力作用下扩展导致的，也可以通过蠕变引入缺陷导致延迟断裂。

显微结构对材料断裂的影响

晶粒尺寸

多晶材料的晶粒越小强度越高。服从霍尔佩奇关系
$$\sigma_f=\sigma_0+K_1d^{-\frac{1}{2}}$$
这是由于晶界导致位错塞积。

气孔

气孔导致应力集中，大多数无机材料的弹性模量和强度都随气孔率增加降低。$\sigma_f=\sigma_0\exp{(-np)}$

高温下，气孔能容纳变形，阻止裂纹扩展，反而提高热稳定性。

第二相

弹性模量较低的第二相和杂质降低强度

无机材料增韧

控制前度的主要参数包括弹性模量$E$,断裂功$\gamma$,裂纹尺寸$c$

c是最容易控制的参数，增韧主要从消除缺陷和阻止缺陷发展入手。

1.微晶、细纤维、晶须
2.预加应力（残余压应力韧化、热膨胀预加应力）
3.化学强化
4.相变增韧，如氧化锆陶瓷
5.弥散增韧：掺入一定量第二相的细粉

6.裂纹偏转增韧

7.裂纹桥接和拔出增韧
